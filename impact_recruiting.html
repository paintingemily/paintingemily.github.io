IMPACT RECRUITING

Responsive Website Development

Impact Recruiting is a recruiting firm located in San Fransisco who came to Design Commission to have their website redesigned to be a more modern and responsive site. During my internship, I was the main developer on this project.

The website needed to be coded with responsiveness and parallax in mind. I was given a basic parallax template to start from. From there, I added markup to the HTML and restyled it with CSS to match the comp. Additional functionality needed to be added to the site with jQuery (carousel, hovers, and on-click reveals).
